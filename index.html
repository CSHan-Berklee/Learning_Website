What is the internet?

Day 2

What is HTML?

The first web page was written in HTML (Hypertext Markup Language). It is the standard markup language used to create webpages. It structures contents on the web, allowing browsers to display text, images, and other elements

How to use HTML?

HTML uses tags to define elements. Tags are enclosed in angle brackets. Elements can include headings. paragraphs, link, images, and lists.
